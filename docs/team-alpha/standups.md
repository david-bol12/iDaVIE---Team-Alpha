# Team Alpha — Cross-sub-team daily stand-up notes

Single shared file for **all 7 sub-teams** in Team Alpha. Required artefact per Section 9.2.6 of the assignment spec.

## Convention

- One `##` heading per **date** (ISO `YYYY-MM-DD`).
- Under each date, one `###` heading per **sub-team**.
- Each sub-team's entry uses the schema below. Keep it terse — bullets, not prose.

```
**Present:** <names>
**Absent:** <names or "none">
**Yesterday:** <what got done>
**Today:** <what's planned>
**Blockers:** <what's stuck, and on whom>
**Cross-team asks:** <requests directed at other sub-teams, by name>
```

Sub-team numbers below follow the **work-package numbering** (Section 6.1–6.7):
1. Architecture / Micro-kernel
2. Persistence & Data
3. Rendering & Compute
4. Interaction System (VR)
5. Networking / Server
6. Desktop GUI & Client Shell *(us)*
7. Native Plug-ins

If you'd rather use cohort allocation IDs (Section 5.5), swap the numbering scheme team-wide — don't mix.
